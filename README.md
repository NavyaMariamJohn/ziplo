# 🔗 ZIPLO – Smart URL Shortener with Analytics

🚀 A full-stack URL shortening platform with advanced analytics, built to provide secure link management and real-time insights.

---

## 🌐 Live Overview

ZIPLO allows users to create short links and track their performance through a modern analytics dashboard.
It combines link management, tracking, and visualization into one clean interface.

---

## ✨ Key Features

### 🔗 URL Shortening

* Generate short, shareable links instantly
* Clean and simple link creation experience

### 📊 Analytics Dashboard

* Total clicks, unique visits, and average clicks/day
* 7-day traffic trend visualization
* Top-performing links with progress indicators

### 🔍 Link-Level Analytics

* Detailed stats for each link
* Daily & weekly click tracking
* 📍 Location-based analytics (pie chart)
* 🕒 Recent click activity (timestamp, IP, location)

### 🔐 Security & Access

* Role-based access (User / Admin)
* Controlled redirection

### ⚡ Performance Optimized

* Smooth and responsive charts
* Optimized rendering (no resize lag)
* Clean and modern UI

## ⚙️ Setup Instructions

### 🔹 Clone Repository

```
git clone https://github.com/your-username/ziplo.git
cd ziplo
```

---

### 🔹 Backend Setup

```
cd backend
pip install -r requirements.txt
python app.py
```

---

### 🔹 Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside backend:

```
SECRET_KEY=your_secret_key
DATABASE_URL=your_database_url
```
