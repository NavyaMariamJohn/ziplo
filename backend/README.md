# ZIPLO - URL Shortener with Analytics

## 📌 Mini Project

ZIPLO is a centralized, role-based URL shortening and analytics system designed for secure and controlled link sharing.

---

## 🚀 Features

- Generate short and unique URLs
- Secure and controlled redirection
- Click tracking and analytics
- Timestamp logging
- IP address tracking
- Approximate geographic location detection
- Role-based access (User / Admin)
- Dashboard for link management
